orientation review

Point this at a folder of new scans. Click any photo to rotate it 90° at a time — click through until it's upright. When you're done, hit save and the corrected files are written straight back to that folder, same filenames, ready to hand to Claude for processing into the site.

This browser can't write files back to your folder directly. Open this page in Chrome or Edge on desktop — Safari and Firefox don't yet support the File System Access API this tool relies on.
